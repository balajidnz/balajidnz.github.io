Privacy

sunair.fun collects nothing about you. That's less a policy than a description of how the site is built — there is nowhere for your data to go.

No tracking, no server

No cookies. No analytics. No ads. No third-party scripts of any kind — a Content Security Policy on every page blocks them outright. Every page here is a static file: there is no database, no login and no back end, so nothing you type is uploaded, because there is nothing to upload it to.

The games

They run entirely in your browser. Some let you write a message and get a link to share. That message is stored inside the link itself — in the part after the #, which browsers, by design, never send to a server. It is saved nowhere. Nobody, including me, can read it. Lose the link and the message is gone.

What I don't control

The site is hosted on GitHub Pages. Like any web host, GitHub's servers see the requests your browser makes — including your IP address — in order to send the page back. That's true of the entire web, it's governed by GitHub's own privacy statement, and I have no access to those logs.
