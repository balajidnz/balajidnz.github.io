sunair.fun

About

sunair.fun is a personal site. I'm Rico, and I make small browser games here in my own time. It isn't a company and it isn't a product.

What's here

Short games you play in a browser tab, in one sitting. Free, no ads, nothing to sign up for, nothing to buy, nothing to install — each game is a web page that runs in your browser and stops existing when you close the tab.

How it's built

Static HTML, CSS and JavaScript on GitHub Pages. No back end, no database, no analytics, no third-party scripts — a Content Security Policy on every page blocks external resources outright. All of it is open source — you can read exactly what these pages do.

Privacy

Nothing here collects anything about you — no cookies, no analytics, no tracking. The details.

Contact

Something broken, a game you liked, or a security vendor who thinks this domain has been miscategorised — hello@sunair.fun. I answer.
